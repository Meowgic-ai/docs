## 用图片构建应用
你可以在对话中上传图片、通过 Visual Editing 替换图片，或使用外部图片 URL 添加素材。这样能让应用 UI 更贴近参考图，也能让界面风格方向更清晰。

**使用外部图片 URL**

你也可以通过外部 URL 引用图片到项目中。该方式适合使用 Unsplash、Imgur 或其他图床平台上的素材。请注意素材版权与授权范围。

## 用参考链接构建应用

添加参考链接可以帮助 Agent 更准确理解你的目标，以及你希望达到的最终效果。
