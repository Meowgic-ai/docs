Meowgic Cloud 是为 AI 驱动电商应用打造的全栈云平台。从用户可见的店铺前端，到背后运行的后端服务，Meowgic 都能自动完成配置与托管。

你只需在 Meowgic 中描述需求，系统就会自动生成应用的前后端，并可直接在云端运行。无需服务器配置、基础设施运维或复杂部署流程。

依托内置托管、数据库、身份认证、存储、边缘函数和 AI 能力，Meowgic Cloud 提供可弹性扩展的计算与存储资源，可随你的应用和业务增长自动扩容。

## 为什么选择 Meowgic Cloud

- **随业务增长平滑扩展**：从第一个电商应用到大规模生产负载，Meowgic Cloud 都能随业务规模与流量增长稳定扩展。
- **默认安全**：企业级身份认证、安全云基础设施、细粒度访问控制与现代安全实践，帮助你保护应用、用户数据和业务流程。
- **成本更可控**：在一个工作区管理多个应用与项目，配合按量计费模型，降低基础设施开销，让上线、验证与扩容更轻松。

## 如何启用 Meowgic Cloud

默认情况下，AI 代理会自动判断你的应用是否需要 Cloud 能力，并在需要时主动提示你开启。你也可以随时手动前往 **App → Cloud → Enable Meowgic Cloud** 启用。

## 区域选择

Meowgic Cloud 项目目前支持部署在以下三个区域：

- Americas
- Europe
- Asia Pacific

选择更靠近用户的区域，通常会带来：

- 更低延迟
- 更快的数据库响应
- 按区域托管数据
- 更好的整体后端性能

<Note>
启用 Cloud 后，所选区域不可更改，现有项目也不能跨区域迁移。
</Note>

## Meowgic Cloud 功能

在任意 Meowgic 项目中，点击 Preview 模式旁的 Cloud 按钮即可进入 Cloud 标签页。

### Database

- **自动生成数据结构**：用自然语言描述应用所需数据，Meowgic 会自动创建数据库表和 schema。
- **无需 SQL 也可管理**：可直接在界面中查看、编辑和整理数据记录，无需写查询语句即可更新字段、检查数据表并保持数据同步。

### Storage

- **零代码文件管理**：可直接上传、管理和分发商品图片、用户上传内容、文档、视频及其他静态资源，无需额外配置后端。
- **存储桶机制**：文件会安全存储在项目专属 bucket 中，并自动生成可在应用中使用的 URL 与引用。
- **容量限制**：单个文件最大支持 1 GB。

### AI

- **内置 AI 能力**：Meowgic Cloud 提供开箱即用的 AI 功能，让你无需额外配置就能打造更智能、更具互动性的应用体验。<Badge color="blue">即将推出</Badge>
- 常见 AI 能力示例：
  - **AI 摘要**：将长文本自动提炼为关键信息
  - **AI 聊天机器人或智能体**：在应用内构建对话式助手
  - **情感识别**：规模化分析用户反馈倾向
  - **图像风格化工具**：将图片转换为不同艺术风格
- 你还可以在 AI 概览页查看项目的 AI 使用情况与性能表现。

### Usage

**Usage** 标签页会以可视化方式展示在所选时间范围内应用对 Meowgic Cloud 资源的使用情况，包括计算、存储、数据库、AI 及其他云服务。

默认时间范围为 **Last 7 days**。你可以在下拉菜单中切换到 **Last day**。Cloud 用量数据每日 CET 时间上午 10 点更新。

用量会按 **资源类别** 拆分，并通过分段条与百分比列表展示：

- **Database server**：应用查询或更新数据时会消耗该资源。可扩展数据服务即将上线。
- **Database storage**：数据库用于存储应用数据（如用户、订单、配置）的磁盘空间。
- **Compute**：应用代码运行时消耗的计算资源。
- **Storage**：用于存储图片、视频、上传文件等资源。
- **Live updates**：用于向用户推送实时更新（如通知或聊天消息）。
- **Network**：应用通过互联网发送或接收数据时消耗的网络资源。

## Cloud 与 AI 按量计费

Meowgic 的所有套餐（含 Free）均提供 Cloud 托管与内置 AI 能力。Cloud 服务采用灵活的按量计费模式，费用会随实际资源使用量变化，并与订阅费用分开结算。

在应用中前往 Settings → Usage，你可以：

- 查看工作区余额状态
- 充值工作区余额（仅付费套餐可用）
- 查看发票历史并管理支付偏好
- 跟踪工作区整体 Cloud 与 AI 使用成本
- 查看各项目的 Cloud 与 AI 详细成本

Cloud 与 AI 余额页面更新频率如下：

- Cloud 用量：每日 CET 时间上午 10 点更新
- AI 用量：持续实时更新

### 充值说明（仅付费套餐）

为 Meowgic Cloud 与 AI 用量充值，仅对付费订阅套餐开放。

按量使用费用独立计费，不包含在订阅费内。

- **Cloud 用量**：取决于应用活跃度（用户数与交互量）以及数据传输量（上传/下载）。
- **AI 用量**：费用取决于所选 AI 模型与任务复杂度。通常模型越高级，资源消耗越高、成本也越高。具体定价请以 Meowgic AI 支持模型章节中的官方链接为准。
